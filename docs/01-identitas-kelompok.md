## Arsitektur Aplikasi

SalonKu merupakan aplikasi manajemen salon berbasis web yang memungkinkan pelanggan melakukan reservasi layanan salon secara online. Sistem juga membantu admin mengelola layanan, jadwal, pelanggan, dan transaksi. Frontend berkomunikasi dengan backend Laravel melalui REST API, sedangkan backend mengelola data pada database MySQL. Deployment dan monitoring aplikasi dilakukan menggunakan Docker dan GitHub Actions.

### Aplikasi 1 — Frontend

- Nama Aplikasi: MySalonGweh Frontend
- Deskripsi Singkat:
  Menyediakan antarmuka bagi pelanggan untuk melihat layanan salon, melakukan reservasi, melihat riwayat booking, dan mengelola profil.
- Berkomunikasi dengan:
  MySalonGweh API (Laravel Backend)

### Aplikasi 2 — Backend (Laravel)

- Nama Aplikasi / Service: MySalonGweh API
- Deskripsi Singkat:
  Menangani autentikasi pengguna, pengelolaan layanan salon, reservasi, pembayaran, dan laporan.
- Menyediakan layanan untuk:
  MySalonGweh Frontend

### Aplikasi 3 — DevOps & Infrastruktur

- Nama Aplikasi / Service: MySalonGweh Deployment Service
- Deskripsi Singkat:
  Mengelola deployment aplikasi, monitoring server, backup database, dan proses CI/CD.
- Menyediakan layanan untuk:
  MySalonGweh Frontend dan MySalonGweh API

# Rencana Fitur

---

## Fitur 1 — Autentikasi Pengguna

**Role Penanggung Jawab:** Backend

**Sumber Data:** Internal System

**Deskripsi & Ekspektasi:**
Memungkinkan pelanggan dan admin melakukan registrasi, login, dan logout. Sistem melakukan validasi data, enkripsi password, serta pengelolaan hak akses pengguna.

---

## Fitur 2 — Booking Layanan Salon

**Role Penanggung Jawab:** Frontend

**Sumber Data:** Internal System

**Deskripsi & Ekspektasi:**
Pelanggan dapat melihat layanan yang tersedia, memilih jadwal, dan melakukan reservasi secara online melalui antarmuka yang mudah digunakan.

---

## Fitur 3 — Manajemen Layanan Salon

**Role Penanggung Jawab:** Backend

**Sumber Data:** Internal System

**Deskripsi & Ekspektasi:**
Admin dapat menambah, mengubah, melihat, dan menghapus layanan salon beserta harga, durasi, dan deskripsinya.

---

## Fitur 4 — Dashboard Admin

**Role Penanggung Jawab:** Frontend

**Sumber Data:** Internal System

**Deskripsi & Ekspektasi:**
Menampilkan ringkasan data seperti jumlah pelanggan, jumlah reservasi, layanan terpopuler, dan pendapatan salon dalam bentuk tabel maupun grafik.

---

## Fitur 5 — Pembayaran Online

**Role Penanggung Jawab:** Backend

**Sumber Data:** Third-Party API — Midtrans

**Deskripsi & Ekspektasi:**
Mengintegrasikan sistem dengan Midtrans untuk memproses pembayaran reservasi secara online dan memperbarui status pembayaran secara otomatis.

---

## Fitur 6 — Riwayat Reservasi Pelanggan

**Role Penanggung Jawab:** Frontend

**Sumber Data:** Internal System

**Deskripsi & Ekspektasi:**
Pelanggan dapat melihat daftar reservasi yang pernah dilakukan, status reservasi, tanggal kunjungan, dan detail layanan yang dipilih.

---

## Fitur 7 — Deployment Aplikasi

**Role Penanggung Jawab:** DevOps

**Sumber Data:** Internal System

**Deskripsi & Ekspektasi:**
Melakukan deployment aplikasi ke server cloud menggunakan Docker dan GitHub Actions sehingga aplikasi dapat diakses secara online.

---

## Fitur 8 — Monitoring dan Backup Sistem

**Role Penanggung Jawab:** DevOps

**Sumber Data:** Internal System

**Deskripsi & Ekspektasi:**
Melakukan monitoring performa server, pengelolaan log aplikasi, serta backup database secara berkala untuk menjaga ketersediaan dan keamanan data.
